# Конвертер СРЗУ на КПТ из xml в html

## Назначение
Преобразовывает xml-файл "Схемы расположения земельного участка или земельных участков на кадастровом плане территории", подготовленный в соответствии с [Xml Shema](https://rosreestr.ru/upload/Doc/10-upr/16_SchemaParcels_V02.rar), утвержденой [приказом](https://rosreestr.gov.ru/upload/Doc/10-upr/п%200152.PDF) Росреестра от 27.03.2017 № П/0152/17 в каталог координат в формате html.

## Способ использования

1. Скачать файл XSL Transformations -- [plan-kpt-xml2html.xsl](./plan-kpt-xml2html.xsl "plan-kpt-xml2html.xsl") и использовать его в своих рабочих процессах.
2. Воспользоваться [онлайн](https://plan-kpt-xml2html.sourceforge.io/plan-kpt-xml2html.xhtml) версией конвертера.
